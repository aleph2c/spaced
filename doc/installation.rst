Installation
============

.. toctree::
   :maxdepth: 2
   :caption: Contents:

To install:

.. code-block:: bash

  python3 -m venv
  . ./venv/bin/activate
  pip install spaced


Spaced has been tested on Python 3.5 and is dependent upon pyyaml, numpy, scipy and
matplotlib.  For testing the package, there are dependencies upon pytest,
pyreadline and Click.
